# StreamX Search

This directory contains the StreamX Search distribution files used by the Adobe Edge Delivery Services project.

The files are copied from the StreamX Search library build output and should not be modified directly.

## Directory structure

```text
search/
├── streamx-search-inline.js
├── streamx-search-results-panel.js
├── streamx-search-tabs.js
├── streamx-search.css
├── README.md
└── eds/
    ├── search-results-panel.js
    ├── search-tab.js
    └── search-tabs.js
```
